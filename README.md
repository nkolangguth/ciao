# Ciao

A mobile-first language learning UI prototype built with React + TypeScript.

## What it demonstrates

- Design tokens for fonts, colors, spacing, and other UI primitives
- Reusable components (e.g. Avatar, IconButton, TutorSection, CallRow)
- Mobile-first layout and navigation (app shell + bottom navigation)
- Interaction patterns such as tutor expand/collapse and tab switching

## Tech stack

- React
- TypeScript
- Vite

## How to run locally

```bash
npm install
npm run dev
```
